# Insight
![Insight Logo](https://imgur.com/a/zOIlAjl)
> Insight is a OSINT tool, which gathers information about user via their usernames


## Installation



```sh
pip install -r requirements.txt
```
Insight Only works with python3

## Usage example

```sh
python insight.py 
```
